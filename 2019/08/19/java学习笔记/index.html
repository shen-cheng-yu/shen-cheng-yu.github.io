概念

实例变量：每个对象都有独特的实例变量，对象的状态由这些实例变量的值决定。

主方法入口：所有的 Java 程序由 public static void main(String []args)方法开始执行。

编码规范

对于所有的类来说，类名的首字母应该大写。如果类名由若干单词组成，那么每个单词的首字母应该大写，例如 MyFirstJavaClass。

所有的方法名都应该以小写字母开头。如果方法名含有若干单词，则后面的每个单词首字母大写。

源文件名必须和类名相同。当保存文件的时候，应该使用类名作为文件名保存（Java 是大小写敏感的），文件名的后缀为 .java。（如果文件名和类名不相同则会导致编译错误）。

标识符

标识符的开头应该是字母，美元符，下划线。开头不能是数字。
首字符之后可以是字母（A-Z 或者 a-z）,美元符（$）、下划线（_）或数字的任何字符组合。

类和对象

类中包含的变量包括：
局部变量：在类的方法中，定义的变量。声明，初始化，作用域都是在这个方法中。
成员变量：定义在类中，方法之外的变量。概念同C++中的成员变量。
类变量：即声明为static的成员变量。

当在一个源文件中定义多个类时，注意只能有一个public类，可以有多个非public类。源文件的名字应该与这个public类的名字一致。

包

包是用来对类和接口进行分类的。

import

import java.io.*导入java.io包下面的所有类。java相当于是一个大包，io相当于是一个小包 ，包的嵌套。

数据类型

内置数据类型

byte short int long数据类型是8/16/32/64位、有符号的，以二进制补码表示的整数。
以byte为例，8位，最小值是-2^7 = -128，最大值是2^7-1 = 127，共可以表示2^8 = 256个整数。

floart double 数据类型是单精度/双精度、32/64位、符合IEEE 754标准的浮点数。浮点数不能用来表示精确的值，如货币。

long float double定义变量的时候，变量后面要加上后缀L f d。

char是一个单一的 16 位 Unicode 字符。\u0000 - \uffff之间。char表示单个字符，用单引号。String表示字符串，用双引号。

boolean表示true/false。

他们取值范围的值都以常量的形式定义在相应的包装类中。java.lang.Byte java.lang.Short java.lang.Integer java.lang.Long java.lang.Float java.lang.Double java.lang.Character。

他们的默认值分别是0 0 0 0L 0.0f 0.0d 'u0000' false。

形如Byte.SIZE 表示该数据类型的位数，Byte.MIN_VALUE 表示该数据类型的最小值，Byte.MAX_VALUE表示该数据类型的最大值。可以直接输出Byte.SIZE 得到这些常量值。

JAVA中还存在另外一种基本类型 void，它也有对应的包装类java.lang.Void，不过我们无法直接对它们进行操作。

引用数据类型

Site site = new Site("Runoob")这里的site就是引用类型。
对象、数组都是引用数据类型。 所有引用类型（包括String字符串）的默认值都是null。

常量

final double PI = 3.1415927;使用final修饰常量。通常使用大写字母表示常量。

当使用常量的时候，前缀 0 表示 8 进制，而前缀 0x 代表 16 进制, 例如：int octal = 0144; int hexa = 0x64;

数据类型转换

满足从低级到高级的数据类型转换。从低到高的顺序是byte,short,char—> int —> long—> float —> double

不能对boolean类型进行类型转换。

把容量大的类型转换为容量小的类型时必须使用强制类型转换。容量小的转换成容量大的会自动进行转换。
