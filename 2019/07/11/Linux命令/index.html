mv移动文件与文件重命名

mv ../a.cpp ./
mv a.cpp abc.cpp #重命名的时候，如果涉及到了路径，就会被当成是移动操作，所以在重命名之前要先进入指定目录

cat查看文件

cat -n ../linux命令.txt |more
-n 显示行号，并且带上管道命令|more来分页浏览

vim编辑器

touch abc.cpp #创建文件
vim abc.cpp #进入vim编辑器，默认是一般模式
i #进入编辑模式
#include<iostream> #编辑文件
按键esc #退出编辑模式，回到一般模式
:wq #:是进入命令模式，w保存文件，q退出文件
g++ -o abc.exe abc.cpp #编译程序
./abc.exe #运行当前目录下的abc.exe这个可执行程序

在一般模式下可以使用的快捷键：

yy 拷贝当前行
5yy 拷贝以下5行
p 小写，在当前行的后面一行粘贴
P 大写，在当前行的前面一行粘贴
:/endl 在命令行下查找关键字,输入n则继续查找下一个
:set nu 设置文件显示行号
:set nonu 取消显示文件行号
u 撤销动作
~ 改变大小写
跳转到指定行:先设置好行号，输入16，按下shift+g，即可跳转
G:跳转到首行
gg:跳转到尾行
dd 删除当前行
5dd 删除以下5行
保存退出：shift+zz(两个大写的z)，:wq
不保存退出：shift+ZQ，:q!（不保存强制退出的意思）
shift+<<和shift+>> 向前向后缩进
:%s/x/y/g 全局将x用y替换掉

获取帮助命令：

man ls
help ls

pwd:获取当前目录的绝对路径

ls

ls -a(all):查看当前目录下的所有文件和目录，包括.当前目录和..上一级目录
ls -l:以列表的形式显示出来(可以简写为 ll)

cd

cd cd~ 返回家目录
cd .. 返回上一级目录
cd /home/xtncsg/桌面 进入一个绝对路径

创建

创建文件:touch abc.cpp
创建目录:mkdir ./hello
创建多级目录: mkdir -p ./hello/nm/tmd

删除

删除文件:rm abc.txt
删除空目录：rmdir hello
删除非空目录: rm -rf abc
删不掉的话，试试加一个 -f 选项
-r表示递归删除，删除该目录下的所有文件以及目录，-f表示强制删除

cp

cp 原文件或目录 目的文件或目录 拷贝文件（Copy）
-r 递归拷贝整个目录

passwd:更改密码

whoami:查看用户id

cal:查看日历

wc filename1 filename2
显示当前文件的总行数，单词数，字节数，文件名
可以一次统计多个文件的信息

标准的Linux流

一般情况下，每个linux程序运行都会创建三个文件流（三个文件）
标准输入流:文件描述符为0，linux程序默认从stdin读取数据
标准输出流:文件描述符为1，linux程序默认向stdout输出数据
标准错误流:文件描述符为2，linux程序会向stderr写入错误信息
这三个流都是文件，都有各自不同的文件描述符。

终端

按下 ctrl+alt+t 进入终端
exit 退出终端

上下方向键可以回到之前输入过的命令

清空

clear:终端内容向下翻开新的一页，终端内容并不消失
reset:清空终端的内容

日期

date:获取当前日期以及时间
date +”格式字符串” 以特定格式输出时间
%Y:四位数的年份 %y:两位数的年份
%D:含有年月日的日期 %d:仅仅含有日期
%H:小时 %M:分钟 %S:秒钟 %m:月份

重复执行某个指令：

watch -n 3 -d -t 需要重复的命令
每隔3秒运行一次，不写则默认2秒。n是interval间隔时间，d是difference，将每一次显示的内容中出现变化的部分高亮显示，t是去除最顶端显示的时间。
比如， watch -n 10 -d -t 'date +"%H:%M:%S"'
