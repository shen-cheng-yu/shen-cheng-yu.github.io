chmod

改变文件的读写和执行权限。

符号法

chmod {u|g|o|a}{+|-|=}{r|w|x} filename
chmod a+rx filename # 让所有用户可以读和执行文件filename。
chmod go-rx filename # 取消同组和其他用户的读和执行文件filename的权限。
其中，u(user，用户自己)，g(group，同组用户)，o(other，其他用户)，a(all，所有用户)。

八进制法

chmod abc file
其中a,b,c各为一个八进制数字，分别表示User、Group、及Other的权限。
|八进制数字|代表的权限|
|:—-:|:——-:|
|4|可读
|2|可写|
|1|可执行|
chmod 741 filename #让本人可读写执行、同组用户可读、其他用户可执行文件filename。
其中，7=4+2+1，三种权限都有的意思。
chmod -r 755 /home/oracle #递归更改目录权限，本人可读写执行、同组用户可读可执行、其他用户可读可执行

通过\$*和\$@

\$* 会将命令行上提供的所有参数当作一个单词保存, 我们得到的值也就相当于是个字符串整体。一个字符串中只有一个单词。
\$@ 会将命令行上提供的所有参数当作同一字符串中的多个独立的单词。一个字符串中有很多个单词。

如果for语句中不加引号，则for默认以空格进行分割，又看不出区别。

字符串

获取字符串长度：

提取子串：

echo ${hello:1} 从某个字符开始，提取之后的所有字符。
echo ${ #hello:1:4}1是子串的起始位置，4是子串的长度。

字符串中查找字符：

echo `expr index "hello" h`hello是要查找的字符串，h是待查找的字符。这里查找到的是从0开始的序号加1的值，即第一个字符查找到的结果是1。
echo `expr index "hello" l0`查找两个字符，哪个字符先出现就显示哪个结果。
